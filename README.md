  # UX
### **User Goals** 
- As a user, I want to answer questions.
- As a user, I want to test my knowledge.
- As a user, I want to beat my score.
- As a user, I want to use the quiz without bugs.
- As a user, I want to be able to view the quiz on any device.

### **Site Owner Goals**
- I want to be able to update the quiz.

### **Returning User**
- I want to beat my score.

# Design
### **Colours**
 - I used "#808283" for the website's background colour.
 - I used "#1EE494" for the background colour of the quiz box.
 - I used "#fff" for the backgrounds of the buttons.
 - I used "#Ff5f2d" for the text of the buttons.
 - I used "#00ff1a" for the background of the page and the button if the user got the question correct.
 - I used "#Ff0000" for the background of the page and the button if the user got the question wrong.
 - I used these colours because the Irish flag is Green, White and Orange.
